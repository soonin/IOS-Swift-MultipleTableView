# IOS-Swift-MultipleTableViewsInOneViewController
``` Level : Intermediate ```

Learn How to create two UITableViews in one view controller using two custom UITableViewCells , Yes Creating multiple tableViews in one ViewController. This tutorial is made with Xcode 10 and built for iOS 12.

## About

Sample app to show TableView inside one viewControler. 

## Requirements

- iOS 12.0+
- Xcode 10.0+
- Swift 4.0+

## Usage

You won't need to Download any extra libaray just ```import UIKit``` inside the project
